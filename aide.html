AIDE : Sommaire

  • 1 - Préparation des fichiers (nouveau sprint)
  • 2 - Initialisation du graph (nouveau sprint)
  • 3 - Créer des branches
  • 4 - Merger

1 - Préparation des fichiers

Dans index.html rajouter au niveau des slides (le X est à incrémenter):

<div class="swiper-slide">
    <h1 class="titre">TITRE DU SPRINT</h1>
    <div id="graph-container-X"></div>
</div>

Ce qui aura pour effet de rajouter une slide avec un conteneur pour le nouveau graphique.


Créer le fichier sprintX.js (X à incrémenter) à la racine.


Le rajouter dans index.html (en bas) :

<script src="sprintX.js"></script>

Il reste plus qu'à mettre à jour régulièrement le fichier sprintX.js (prochaines slides -> )

2 - Initialisation du graph

Dans le fichier sprintX.js, instancier le graph dans le conteneur : (incrémenter le X comme précédemment)

var sprintX = creerGitGraph("graph-container-X");

Puis créer la branche MASTER :

var master = creerBrancheMaster(sprintX);


Puis créer la branche INTEGRATION :

var integration = creerBrancheIntegration("integration_app_Y");

3 - Créer des branches

FONCTIONNALITE

Dans le fichier sprintX.js, à la suite de la branche d'intégration :

var fonctionnalite1 = creerBrancheDeFonctionnalite("f_nom_de_la_fonctionnalite");



TACHE

Dans le fichier sprintX.js, à la suite de la branche de fonctionnalité associée à la tache :

var tache1 = creerBrancheDeTache(fonctionnalite1, "0000_nom_de_la_tache");



SPRINT

Dans le fichier sprintX.js, à la suite de toutes les branches de fonctionnalités et de taches :

var brancheSprint = creerBrancheDeSprint("twenty_r00_s00");



SPIKE

Dans le fichier sprintX.js, si une branche de spikes est à créer, la créer après la branche de sprint :

var brancheSpikes = creerBrancheDeSpikes("spikes_twenty_r00_s00");

4 - Merger

Dans le fichier sprintX.js, aprés l'initialisation des branches :

MERGE DE TACHE SUR FONCTIONNALITE

fonctionnalite1.merge(tache1);



MERGE DE FONCTIONNALITE SUR BRANCHE DE SPRINT

brancheSprint.merge(fonctionnalite1);



MERGE DE FONCTIONNALITE SUR BRANCHE D'INTEGRATION (fusion après démo)

creerMerge(integration, fonctionnalite1);
